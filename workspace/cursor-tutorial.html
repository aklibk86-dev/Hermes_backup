⚙️ 第二步：打开 Cursor 设置

  1. 打开 Cursor
  2. 点击左下角齿轮图标 ⚙️ → Settings
  3. 或按快捷键 Cmd/Ctrl + Shift + J 直接打开
🖼️

【这里放截图2：Cursor 设置页面入口位置】

🔑 第三步：找到 Models 配置项

  1. 在 Settings 左侧菜单找到 Models
  2. 往下滑到底部，找到 Override OpenAI Base URL
  3. 填入你的中转地址，例如：https://api.wf1.one/v1
🖼️

【这里放截图3：Models 设置页面的 Override OpenAI Base URL 输入框】

🚀 第四步：填入 API Key 并保存

  1. 在同一页面找到 API Key 输入框
  2. 填入你的中转 Key（sk- 开头的字符串）
  3. 点击 Save 保存配置
  4. 重启 Cursor 即可生效 🎉

❓ 常见问题

Q：配置了之后 Cursor 自带的模型还能用吗？

A：能。配置了中转地址之后，你既可以用官方模型（如果还有额度），也可以用中转的模型，两者不冲突。

Q：中转的 Key 安全吗？

A：中转服务只转发你的请求到官方 API，不会存储你的对话内容。选择靠谱的服务商就行。

Q：速度会比官方慢吗？

A：香港节点直连，延迟很低。实际体验和官方几乎没有差别。中转还能走 CDN 加速，部分地区反而更快。

Q：用中转会被封 Cursor 账号吗？

A：不会。Cursor 官方就提供了这个设置项，是合法的配置方式。你只是改了 API 地址而已，跟改代理地址是一个意思。
