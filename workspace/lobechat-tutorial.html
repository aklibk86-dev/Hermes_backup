🐳 方式一：Docker 部署（推荐）

这种方式最适合自己有服务器/VPS 的人，一条命令搞定：

docker run -d -p 3210:3210 \
  -e OPENAI_API_KEY=你的中转Key \
  -e OPENAI_PROXY_URL=https://api.wf1.one/v1 \
  --name lobe-chat \
  lobehub/lobe-chat

运行之后，浏览器打开 http://你的IP:3210 就能用了。

💡 小提示：如果你有域名，可以用 Nginx/OpenResty 反代，加上 HTTPS，体验更好。

▲ 方式二：Vercel 一键部署（免费）

没有服务器也能部署，Vercel 提供免费额度：

  1. 访问 LobeChat 官方部署链接（GitHub 仓库 README 里有 Vercel 按钮）
  2. 点击 Deploy with Vercel，用 GitHub 账号登录
  3. 在环境变量设置页面填入：
环境变量名 填写值
OPENAI_API_KEY 你的中转 Key
OPENAI_PROXY_URL https://api.wf1.one/v1
ACCESS_CODE 自己设一个访问密码（可选）

部署完自动获得 https://xxx.vercel.app 域名，可以直接用。

⚙️ 进阶配置：接入多个模型

一个中转 Key 可以同时支持多个模型，LobeChat 默认会自动读取。

如果你想让不同模型走不同的 Key，可以在环境变量里加：

# 主 Key（默认模型）
OPENAI_API_KEY=sk-xxxxx
OPENAI_PROXY_URL=https://api.wf1.one/v1

# Claude 单独配置
ANTHROPIC_API_KEY=sk-xxxxx
ANTHROPIC_PROXY_URL=https://api.wf1.one/v1

LobeChat 自动识别多个 Provider，在界面上你可以随便切换。

❓ 常见问题

Q：LobeChat 跟 ChatGPT 网页版比有什么优势？

A：最大的优势是「自由」。你可以随时切换不同模型做对比，内置了插件系统（联网搜索、画图、图表），还有知识库功能，可以把文档上传给它让它学习。这些 ChatGPT 网页版都不具备。

Q：Docker 部署需要什么配置？

A：要求很低，1GB 内存的服务器就够。LobeChat 本身只是一个前端界面，实际的 AI 推理在你中转 API 那边跑。

Q：Vercel 免费版够用吗？

A：够用。Vercel 免费版每月 100GB 带宽和 100 小时运行时长，个人日常使用绰绰有余。

Q：部署完了但是连不上模型，报错怎么办？

A：检查两个地方：1) API 地址末尾是不是有 /v1；2) API Key 有没有填对。大部分问题都是这两个原因。还有问题可以直接问我。
